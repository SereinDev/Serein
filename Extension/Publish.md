
# 发布

>[!TIP]如果你有好的插件或正则、任务文件，你也可以[提交ISSUE](https://github.com/Zaitonn/Serein/issues/new/choose)/[PR](https://github.com/Zaitonn/Serein/compare)或与作者联系后即可发布在这里

提交时记得附加以下内容：

- 作者
  - 联系方式
  - 名称
  - *交流群 (可选)*
- 插件
  - 名称
  - 简介
  - 功能
  - 使用方法
  - *图片 (可选)*
- 永久下载 / 源文件

如果你会`markdown`的语法，你也可以自己写一份详细的插件文档通过联系作者发布在这里\\(￣︶￣*\\))
