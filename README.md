## ▶ 运行环境

|                     | Serein.Cli | Serein.Lite | Serein.Plus |
| ------------------- | :--------: | :---------: | :---------: |
| .NET 运行时 8.x     |    需要    |    需要*    |    需要*    |
| .NET 桌面运行时 8.x |            |    需要     |    需要     |

_*在下载时，`桌面运行时`包含`运行时`_

## ⛏ 编译

[.NET SDK 8.0+](https://dotnet.microsoft.com/zh-cn/download/dotnet)

```sh
dotnet build
```

或

```ps
.\src\Scripts\publish.bat
```
